The Performance tab provides detailed role-based performance analysis and identifies the most successful player combinations for your team.

## Features

### Role Performance Card

![Role Performance Card](/role-performance.png)

Comprehensive statistics for each role (Tank, Damage, Support) displayed in a three-column layout:

#### Metrics Displayed:
- **Playtime**: Total time played in the role
- **Maps**: Number of maps played
- **K/D Ratio**: Final blows to deaths ratio
- **Damage per 10 Minutes**: Average damage output normalized to 10-minute intervals
- **Healing per 10 Minutes**: (Support only) Average healing output normalized to 10-minute intervals
- **Deaths per 10 Minutes**: Average deaths normalized to 10-minute intervals
- **Ultimate Efficiency**: Eliminations per ultimate used
- **Raw Statistics**: Total eliminations, deaths, and assists

#### Visual Indicators:
- Color-coded K/D ratios (green for ≥1.0, red for \<1.0)
- Color-coded death rates (green for \<5 per 10min, red for \>10 per 10min)
- Role-specific color themes (Tank: blue, Damage: red, Support: yellow)

### Best Role Trios Card

![Best Role Trios Card](/best-role-trios.png)

Identifies the top-performing 5-player combinations (1 Tank, 2 Damage, 2 Support) based on win rate:

#### Features:
- **Ranked List**: Top 5 trios sorted by win rate
- **Win Rate Display**: Color-coded badges (green ≥60%, blue ≥50%, red \<50%)
- **Game Count**: Number of games played with each trio
- **Win-Loss Record**: Detailed breakdown for each combination
- **Expandable Details**: Click to view full roster composition:
  - Tank player
  - Two Damage players
  - Two Support players
- **Minimum Threshold**: Only shows trios with 3+ games played

#### Use Cases:
- Identify which player combinations work best together
- Plan roster rotations based on proven success
- Understand team synergy patterns
- Make data-driven decisions about lineup changes

## Key Insights

The Performance tab helps answer questions like:
- Which role is performing best/worst?
- What are our most successful player combinations?
- How efficient is our ultimate usage per role?
- Which trios should we use more frequently?

## Data Requirements

- Role Performance: Requires player statistics from scrims
- Best Role Trios: Requires at least 3 games with the same 5-player combination
