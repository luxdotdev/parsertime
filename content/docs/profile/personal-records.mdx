---
title: "Personal Records"
---

Showcases the player's best individual achievements across all heroes and scrims.

![Personal Records](/personal-records.png)

## Tracked Records

**Highest SR**
- Best skill rating achieved on any hero
- Shows which hero achieved this rating

**Best K/D Ratio**
- Best elimination-to-death ratio in a single scrim
- Displays the actual K/D numbers and hero used

**Most Eliminations**
- Highest elimination count in a single scrim
- Shows hero played during this performance

**Most Healing**
- Highest healing output in a single scrim (Support heroes)
- Displays hero used

**Most Damage**
- Highest damage dealt in a single scrim
- Shows hero played

**Best Accuracy** (if applicable)
- Highest weapon accuracy percentage achieved
- Displays hero used (typically relevant for aim-intensive heroes)

**Total Playtime**
- Combined playtime across all heroes
- Shows number of different heroes played

Each record card displays:
- Colored icon representing the stat type
- The record value in large, bold text
- Secondary information (hero name, specific stats)
- Hover effect for interactive feel
